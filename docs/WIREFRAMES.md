| Transition | Duration | Easing | Usage |
|------------|----------|--------|-------|
| Screen push | 300ms | ease-out | Navigate forward |
| Screen pop | 250ms | ease-in | Navigate back |
| Modal open | 300ms | spring | Show modal |
| Modal close | 200ms | ease-in | Hide modal |
| Bottom sheet | 350ms | spring | Gift panel, etc. |

| Element | Animation | Duration |
|---------|-----------|----------|
| Button press | Scale to 0.98 | 100ms |
| Button release | Scale to 1.0 | 150ms |

| Checkbox | Scale + checkmark draw | 200ms |
| Input focus | Border glow | 150ms |

| Feature | Animation Type | Notes |

| Queue search | Radar ripple | Continuous |
| Match found | Confetti + photos slide | 3 seconds |
| Gift sent | Gift flies to partner | 1 second |
| Rating stars | Pop in sequence | 500ms total |
| Badge earned | Shine sweep | 1 second |
